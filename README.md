# React Exchange

[Example](http://react-exchange.surge.sh/)

## Description:

React/redux based currency converter.

## Tech Stack:

- ES6
- React
- Redux
- Webpack
- npm

## Getting Started

### Installing

`git@github.com:legan-/react-exchange.git`
\
`cd react-exchange`
\
`yarn`

## Running app:
`yarn dev:watch` and then go to http://localhost:3000
