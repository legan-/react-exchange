# React Exchange

React/redux based currency converter.

## Tech Stack:

- ES6
- React
- Redux
- npm

## Getting Started

### Installing

To prepare this app for running you have to:

1. Install **node.js**
2. `git clone git@github.com:legan-/react-exchange.git`
3. `cd react-exchange`
4. `yarn install`

### Running app

`yarn start`
